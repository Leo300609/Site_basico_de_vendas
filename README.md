# 🌐 hDC Host - Landing Page de Hospedagem

Projeto de uma *landing page* profissional para uma empresa de hospedagem de sites (hDC Host). O foco deste projeto foi criar um layout responsivo, utilizando conceitos de Flexbox e adaptação para diversos tamanhos de tela.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica, organização de seções (navbar, banner, serviços, preços, domínios, contato) e uso de ícones via FontAwesome.
* **CSS3:** * **Flexbox:** Utilizado para alinhar itens na navbar e criar a grade de planos.
    * **Responsividade:** Uso de *Media Queries* para adaptar o site em dispositivos móveis (tablets e celulares).
    * **Estilização de Elementos:** Efeitos de hover em botões, transições suaves e design focado em UX.

## 🧠 Principais Aprendizados:
* **Layout Responsivo:** Ajuste de containers e elementos para diferentes *viewports* (telas de 1100px, 900px e 576px).
* **FontAwesome:** Integração de bibliotecas externas de ícones para compor a interface.
* **Sessão de Preços:** Criação de uma tabela de preços comparativa com destaque visual para o plano "Dedicado Plus".
* **Formulários:** Estruturação de campos de busca de domínio e formulário de contato funcional para o usuário.

## ⚙️ Como visualizar
1. Clone este repositório.
2. Certifique-se de ter as pastas `img/` e `css/` organizadas.
3. Abra o arquivo `index.html` no seu navegador.
